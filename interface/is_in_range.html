public interface is_in_range

Test whether a value lies within the inclusive range [lo, hi].

Returns .true. when lo <= x .and. x <= hi. Supported types are: integer(int32), integer(int64), real(sp), real(dp). Usage: flag = is_in_range(x, lo, hi)
